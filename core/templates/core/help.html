Staff

Name and phone number of newly registered staff are required.

Take care to enter staff names correctly the first time before selecting them when recording jobs because updating the names afterwards may make it difficult to search for jobs using their former names.

After deactivating a staff, close and restart the program for the changes to be reflected throughout the site if it does not do so immediately, then refresh the page.

Deactivated staff will not show up as options for staff assigned to a job.

Inventory

The warranty periods of the inventory is not required for each inventory. Batteries and inverters should have warranty periods so that a reminder appear when the product is out of warranty.

If an inventory is no longer required, update the inventory record by selecting Deactivate.

After deactivating an inventory, close and restart the program for the changes to be reflected throughout the site if it does not do so immediately, then refresh the page.

If you deactivate an inventory, it will no longer show up as an option when recording sales or jobs until it is activated again.

Sales

Sales must contain goods sold, including the unit price and quantity sold.

If no date is specified, the current date is taken as default.

Jobs

Jobs must contain staff assigned, date, nature of the job and transport fare. If no date is specified, the current date is taken as default.

Report

Sales and jobs can be filtered by year, month, staff assigned, goods sold, location, transaction type, and by any combination.

When filtering by month, although the year appears beside the month, only the month is considered.

When the 'Export to Excel' option is selected before filtering, an MS Excel file is created which contains the report of the sales and jobs as specified by the filter parameters. The file is saved in the Documents folder.

Reminders

A battery refill reminder will appear 6 months after the job or sale occured in which the batteries were sold.

A maintenance reminder will appear 12 months after the panel installation.

An expired warranty reminder will appear after the warranty period of the battery has passed.

When a Battery Refill reminder is 'handled', the last refill date of the batteries in the sale or job is updated to the current date. The reminder will show up again in 6 months.

When a Maintenance reminder is 'handled', the last maintenance date of the panel installation is updated to the current date. The reminder will show up again in 12 months.

Any reminders that are 'forgotten' will not be shown again. Only select this if the reminder will never be necessary in future.
